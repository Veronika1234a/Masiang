# MASIANG Website Purpose

## Website Ini Untuk Apa?
Website MASIANG adalah platform pendampingan pendidikan untuk menghubungkan:
- Sekolah (pengajuan dan monitoring pendampingan)
- Pengawas/pendamping (proses bimbingan dan evaluasi)
- Admin (monitoring dokumen, validasi, dan kontrol operasional)

Tujuan utamanya adalah membuat proses pendampingan sekolah menjadi:
- Terstruktur
- Transparan
- Berbasis data
- Berkelanjutan

## Masalah yang Diselesaikan
Sebelum ada platform seperti MASIANG, proses pendampingan sering tersebar di chat, dokumen manual, dan arsip terpisah.
MASIANG menyatukan alur tersebut agar:
- Pengajuan tidak tercecer
- Status kegiatan mudah dilihat
- Dokumen tersimpan terpusat
- Riwayat pendampingan mudah ditelusuri

## Alur Pengguna Utama
1. Sekolah melihat landing page dan mendaftar akun.
2. Sekolah login lalu masuk ke dashboard.
3. Sekolah membuat booking pendampingan.
4. Sekolah memantau status tahapan pendampingan.
5. Sekolah mengakses riwayat dan dokumen hasil pendampingan.
6. Admin melihat seluruh dokumen, detail dokumen, dan ringkasan aktivitas.

## Ruang Lingkup Implementasi Saat Ini
Fase saat ini fokus pada:
- UI/UX frontend
- Navigasi antar halaman
- Validasi form dasar di sisi client
- Struktur dashboard user dan admin

Belum termasuk:
- Integrasi API backend
- Autentikasi sesungguhnya (token/session)
- Penyimpanan data ke database
- Upload/download file real
